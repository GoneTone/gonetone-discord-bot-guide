# 設定用來顯示成員離開訊息的群組頻道

# 指令

注意

  • 只能在群組內執行此指令。
  • 成員需要 管理伺服器 權限才可執行此指令。
/settings memberleave setchannel [channel]

/settings memberleave setchannel #joinleave-gonetone-bot-dev-test
參數名稱類型說明必要
channelChannel選擇文字頻道 (如未選擇頻道，則設定為當前輸入指令的頻道)

# 結果

# 指令

注意

  • 只能在群組內執行此指令。
  • 成員需要 管理伺服器 權限才可執行此指令。
/settings memberleave set

提示

訊息可插入符號：

  • {member} 成員名稱
  • {guild} 群組名稱
