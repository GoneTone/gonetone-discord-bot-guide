## 查詢原神角色資訊

### 指令

:::: code-group
::: code-group-item 格式
```text:no-line-numbers
/genshin character [name] [?vision]
```
:::
::: code-group-item 範例
```text:no-line-numbers
/genshin character 夜蘭
```
:::
::::

| 參數名稱   | 類型     | 說明                               | 必要  |
|--------|--------|----------------------------------|:---:|
| name   | String | 角色名稱                             |  V  |
| vision | String | 角色元素<br>- 如果角色有多種元素才需要輸入 (例如旅行者) |     |
